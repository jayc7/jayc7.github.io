Project TwoNote

Description

Project TwoNote is a note taking web app my good friend and I decided to build in the summer of 2018. In the process of doing so, we learned how to use the PyGobject API
JavaFX was used to implement the front end portion of the project and a series of algorithms constitutes the back end.
The goal of the game is to combine identical tiles until a tile reflects 2048.


Team: Jason Cabrera, Jorge Aparicio

Language: Python/PyGobject
